How to Run
1
Clone the Repository
Open terminal and run git clone <repo-url>, then cd into the project folder.
2
Build & Start Containers
Run docker compose up --build to build images and start all services.
3
Access the Application
Navigate to localhost:3000 in your browser once containers are running.
4
Inspect Networks & Volumes
Use docker network inspect and docker volume ls to verify setup.
5
Tear Down
Run docker compose down -v to stop and clean up all containers and volumes.
